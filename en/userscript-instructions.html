Instruções para adicionar o userscript

O userscript é a versão do Burlesco para ser usado com o TamperMonkey. Funciona no Chrome, Edge, Opera e Firefox. Todos os sites suportados pela extensão também funcionam com o userscript.

Seguem as instruções:

  1. Instale a extensão Tampermonkey
  2. Adicione o burlesco.user.js
  3. Limpe o cache do seu navegador (instruções)
  4. Pronto! Agora é só aproveitar o Burlesco.

Esse método é o preferido para uso no Chrome, já que a instalação manual da extensão não possui atualizações automáticas.
