Instruções para usar o Burlesco no Chrome

O Google removeu o Burlesco da Chrome Web Store por considerá-lo uma violação das suas diretrizes. Há duas formas de contornar esse problema.

O método recomendado é a instalação do userscript para TamperMonkey, que possui atualizações automáticas. As instruções para instalar o userscript podem ser encontradas aqui.

O segundo método é a instalação manual da extensão. O problema é que você reberá avisos pedindo para desativar o Modo do desenvolvedor sempre que iniciar o Chrome no Windows. Além disso, você não receberá atualizações automáticas.

Seguem as instruções para a instalação manual:

  1. Baixe o arquivo burlesco-chrome.zip
  2. Descompacte
  3. Digite chrome://extensions/ na barra de endereços do Chrome
  4. Ative o Modo do desenvolvedor no canto superior direito
  5. Clique em Carregar extensão expandida...
  6. Selecione a pasta onde o burlesco-chrome foi descompactado
  7. Pronto! Agora é só aproveitar o Burlesco.

Se você usa navegadores baseados no Chromium, também pode instalar a extensão ao ir em chrome://extensions e arrastar o burlesco-chromium.crx para dentro da página.

Como remover o aviso no Windows?

Há algumas alternativas para quem deseja utilizar o Burlesco no Chrome (Windows) sem os avisos pedindo para desativar o Modo de desenvolvedor.

Você pode instalar o Chrome Canary, que é a versão experimental do Chrome. Trata-se de uma distribuição que não está totalmente pronta para produção e pode conter bugs, mas não exibe os avisos.

Se preferir usar a versão estável, é possível remover o aviso editando o Group Policy do Windows, que é um processo um pouco trabalhoso, porém permanente.

Outra opção é utilizar o Chromium. Infelizmente, os executáveis para Windows não são distribuídos oficialmente pelos desenvolvedores do Chromium, mas há executáveis preparados por terceiros que podem ser encontrados facilmente.
